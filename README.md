# Overview
This repository contains usages of various Lombok annotations.

Examples of following annotations are covered:
* `@Log`
* `@Cleanup`
* `@NonNull`
* `@ToString`
* `@SneakyThrows`
* `@Getter` and `@Setter`
* `@EqualsAndHashCode`
* `@Data`
* `@Value`
* `@Builder`


For more details visit the [official documentation.](https://projectlombok.org/)
